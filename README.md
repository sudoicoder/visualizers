# Visualizers

Visualizers is a React application for visual demonstration of various algorithms

## Available Visualizers

### 1. Sorting Visualizer

- Selection
- Bubble
- Insertion
- Quick
- Merge

### 2. Path Finding Visualizer

- Dijkstra
